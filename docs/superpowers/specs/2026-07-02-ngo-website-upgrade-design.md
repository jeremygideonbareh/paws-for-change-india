# Paws for Change India — Website Upgrade

> **Goal:** Transform the single-page NGO site into a full multi-page website with content, forms, donations, error monitoring, and design polish — all at $0 cost.

**Architecture:** Static export on GitHub Pages. Content as TypeScript data files. Web3Forms for contact. Mailchimp embed for newsletter. Razorpay for donations. Sentry free tier for error monitoring.

**Tech Stack:** Next.js 16 (static export), React 19, Tailwind v4, Framer Motion 12, TypeScript

## Phases

### Phase 1: Content & Functionality (current)
7 new pages, working contact form, newsletter, Razorpay donations, replace all placeholder links

### Phase 2: Design Polish (future)
Dark mode, page transitions, micro-interactions, accessibility

### Phase 3: Performance & Error Monitoring (future)
Sentry integration, GSAP→Framer Motion swap, bundle optimization
